# MyZone

**MyZone** is an AI-powered civic issue reporting and community management platform built to help residential communities report, triage, track, and resolve local infrastructure issues efficiently.

The platform combines **Google Gemini AI**, **React**, **TypeScript**, and **Express** to provide intelligent issue validation, automated triage, duplicate detection, transparent tracking, and an administrator dashboard.
